# 酷猫音乐天气 - MUI HTML5应用

基于MUI框架的天气查询与音乐播放混合应用

## 功能特点

### 首页
- 城市天气查询（支持任意城市）
- 实时天气数据显示
  - 当前温度
  - 天气状况
  - 湿度
  - 风速
  - 能见度
- 未来三天天气预报
- 悬浮音乐播放卡片

### 音乐播放
- 唱片旋转动画效果
- 播放进度条控制
- 上一首/下一首切换
- 播放列表管理
- Toast提示反馈

## 技术栈

- MUI框架（移动端UI框架）
- Font Awesome图标库
- 网易云音乐API（歌曲播放）
- WeatherAPI（天气数据）
- wttr.in API（备用天气）

## 运行方式

### 方式一：直接在浏览器中打开
```bash
# 直接打开HTML文件
open index.html
```

### 方式二：使用本地服务器
```bash
# Python 3
python -m http.server 8080

# 然后访问 http://localhost:8080
```

### 方式三：打包为APK
使用DCloud HBuilderX：
1. 新建项目 -> 5+App
2. 将代码导入项目
3. 运行到真机/模拟器
4. 云打包生成APK

## 天气接口

- 主接口：https://wttr.in/{城市}?format=j1
- 备用接口：https://api.weatherapi.com/v1

## 注意事项

1. 部分音乐链接可能因版权问题无法播放
2. 天气API可能存在跨域限制，建议在真机上测试
3. 使用模拟器测试时，可能需要配置代理解决跨域问题
4. 建议使用Chrome DevTools的移动端模式进行调试
